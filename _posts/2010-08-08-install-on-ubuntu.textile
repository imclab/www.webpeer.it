neoip-webpack is simple to install on ubuntu.
<pre class="terminal"><code>$ sudo add-apt-repository ppa:jerome-etienne/neoip && \
	sudo apt-get update && sudo apt-get install neoip-utils
</code></pre>

Check it is all ok
<pre class="terminal"><code>$ neoip-detect 
webpack is installed and uptodate.
</pre>

Use it with mplayer
<pre class="terminal"><code>$ mplayer `neoip-url-static http://mirror.bigbuckbunny.de/peach/bigbuckbunny_movies/big_buck_bunny_1080p_surround.avi`
</pre>
