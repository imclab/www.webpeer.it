I just released a node.js version of webpeer binding. Those bindings give easy access to webpeer feature from any nodejs code. There is 2 versions of the binding, <a href="/blog/2010/08/09/webpeer-binding-for-the-web">one for the web</a> and one <a href="/blog/2010/08/24/webpeer-binding-for-nodejs">one for nodejs</a>. This is the same code as they are both javascript, only the paint changes.

Lets install webpeer module for nodejs with the excelent <a href="http://github.com/isaacs/npm">npm</a> packager.

{% highlight bash %}
npm install webpeer
{% endhighlight %}

This is a simple example which probe if webpeer is present, and then convert an url.

{% highlight javascript %}
var webpeer	= require('webpeer');
webpeer.ready(function(){
	console.log("Webpeered url is "+webpeer.url('http://example.com/file.mp4');
});
{% endhighlight %}
