Webpeer Http Proxy

24 Aug 2010

This is a little experiment to use webpeer in a http proxy. Why ? because WebPeer is quite flexible in location and an http proxy is the main part of installing webpeer on a middlebox. If it is installed on a ASDL modem, all the computers behind it will transparently benefit from it. OK, lets get to it.

First lets install nodejs binding for webpeer.

npm install webpeer

Then download webpeer-http-proxy. It is quite short. webpeer-http-proxy will intercept all http requests, and reroute the audio/video files thru webpeer. Lets run it

node webpeer-http-proxy.js

Point your browser or any http client to it. Depending on the OS, it is done differently. On ubuntu, just export an http_proxy environment variable, your milage may vary.

export http_proxy=http://127.0.0.1:8080

Then you can launch your browser or any http client to it.

mplayer http://example.com/movie.mp4
