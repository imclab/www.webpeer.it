Install on Ubuntu

08 Aug 2010

neoip-webpack is simple to install on ubuntu.

$ sudo add-apt-repository ppa:jerome-etienne/neoip && \
	sudo apt-get update && sudo apt-get install neoip-utils

Check it is all ok

$ neoip-detect 
webpack is installed and uptodate.

Use it with mplayer

$ mplayer `neoip-url-static http://mirror.bigbuckbunny.de/peach/bigbuckbunny_movies/big_buck_bunny_1080p_surround.avi`
