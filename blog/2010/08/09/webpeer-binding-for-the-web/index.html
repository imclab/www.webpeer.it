Webpeer Binding For The Web

09 Aug 2010

WebPeer has now usable from any webpage. The source is available here. The API has been kept simple and documented.

As an example, here is a little script to webpeerize all the images in your page. It would be as simple to webpeerize flash <object> or video <source>.

<script type="text/javascript" src="http://webpeer.it/js/webpeer.min.js"></script>"
<script>
	// detect webpeer presence
	webpeer.ready(function(){
		// display presence in js console
		console.dir("webpeer is "+(webpeer.present() ? "present" : "not present"));
		// webpeerify all <img> href
		var elements	= document.body.getElementsByTagName("img");
		for(var i=0; i < elements.length; i++ ){
			elements[i].src	= webpeer.url(elements[i].src);
		}
	});
</script>

Now your images will be downloaded with p2p efficiency. Pretty simple no ?
