Example From nodejs

Lets install webpeer module for nodejs with the excelent npm packager. {% highlight bash %} npm install webpeer {% endhighlight %}
This is a simple example which probe if webpeer is present, and thenconvert an url. {% highlight javascript %} var webpeer = require('webpeer'); webpeer.ready(function(){ console.log("Webpeered url is "+webpeer.url('http://example.com/file.mp4')); }); {% endhighlight %}
