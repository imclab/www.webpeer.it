  • Webpeer seamlessly unifies web and p2p to leverage the best both worlds in a bris. It is efficient , robust and transparent .

  • Efficient

    Webpeer brings p2p efficiency to the web. People in the web peer cloud collaborates with each other. Their sharing allows a reduced server load and a faster downloads for the client.

  • Robust

    Webpeer leverages content from multiple sources. This redundancy allows no single point of failure, that you can see in usual http client/server model. Thus you enjoy more reliable content delivery.

  • Transparent

    Webpeer is transparently integrated to your web environnment without changing it. It has been designed to be non-intrusive. It is transparent for content. Use your unmodified files and streams now! It is transparent for software. Keep using your usual http clients and servers.
